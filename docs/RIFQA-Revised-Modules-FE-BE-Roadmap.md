# RIFQA Revised Modules, Features, FE/BE Roadmap

The full source roadmap lives in the local strategy folder:

`RIFQA/RIFQA App/RIFQA-Revised-Modules-FE-BE-Roadmap.md`

This repository implements the first frontend slice from that roadmap:

- Arabic-first RTL app shell
- Light mode and dark mode
- Home dashboard
- Pregnancy timeline
- Daily check-in
- AI companion
- Mental health support

Core FE non-negotiables:

1. Arabic is the primary product language.
2. Light mode and dark mode must be designed as equal first-class experiences.
3. Mental-health and crisis flows must remain calm, private, and paywall-free.
4. Pregnancy table-stakes tools must be visible in the product architecture from the beginning.
5. The UI should feel Saudi-native, warm, and premium, not like a translated Western tracker.
